# My pong

In this project I used the SFML library and C++ to create a Pong game. The topics I learned are the following:

* Objected oriented programing,
* 2D physics
* 2D collision detection

This idea comes from the Austin Morlan project. I tried to use a different library (SFML instead of SDL2) and I implemented certain things differently. Moreover I added
the game menu.
